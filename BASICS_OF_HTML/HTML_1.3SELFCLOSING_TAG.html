HTML 1.3 SELF CLOSING TAGS

Self closing tags like <hr /> or <br /> are used within other tags, like <p></p>
where you want to separate text without using multiple paragraph tags.
<hr /> -> Horizontal Rule used to seperate texts with horizontal line in between
<br /> -> move the after text to the next line
