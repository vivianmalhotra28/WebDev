CSS 2.1 INTEGRATING CSS

CSS stands for CASCADING(Follows the hierarchy) STYLE SHEET is directly interpret by the browser where as SCSS, Syntactically Awesome Style Sheet uses a preprocessor SASS that processes the .scss to standard css file.

There are three ways through which we can integrate the CSS to HTML

  1. Inline CSS -> with html element we use style attribute
    e.g. <html style="Background:blue">
    </html>
  2. Internal CSS (used for a single webpage)-> used in the <head> </head> of the HTML doc.
    e.g. <head>
    <style>
    html{
    background:red;
    }
    </style>
    </head>
  3. External CSS (Multiple Webpage)-> referred .css file within HTML doc.
    e.g. <head>
    <link
    rel="stylesheet"
    href="./<filename>.css"
    />
    </head>
