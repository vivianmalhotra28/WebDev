CSS 2.3 FONT PROPERTIES

  1. FONT SIZE: Size of letters in an element.
    SYNTAX:
    selector { <-(Id, class, element, etc.)
    font-size:20px;
    }
    • 1 px = (1/96)th inch -> I'm 20px
    • 1 pt = (1/72)th inch -> I'm 20pt
    • 1 em = 100% of parent -> I'm 2em
    • 1 rem = 100% of root -> I'm 2rem{Root is the <html></html>}

  2. FONT WEIGHT: Makes the font heavier.
    SYNTAX:
    selector { <-(Id, class, element, etc.)
    font-weight:bold;
    }
    • Normal Bold: keywords like bold, bolder
    • Lighter Bolder: Relative to parent +/-100 lighter or bolder
    • Numbers: 100-900
  3. FONT FAMILY: Style of text.
    SYNTAX:
    selector { <-(Id, class, element, etc.)
    font-family:Helvetica,sans-serif;
    }
