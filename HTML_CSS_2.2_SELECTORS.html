CSS 2.2 SELECTORS

SYNTAX:
css selector{
property:value;
}

  1. Class Selector: Class is an case-sensitive attribute used with HTML Element to gave css property to certain elements
    Syntax:
    <head>
    <style>
    .red_font{ <--- dot<class_name>
    color:red;
    }
    </style>
    </head>

    <body>
    <p class="red_font">Hello Mr. DJ</p>
    <p>How are you?</p>
    </body>

    RESULT:

    Hello Mr. DJ

    How are you?


  2. ID Selector: Id is an case-sensitive attribute used with HTML Element to gave css property to one element in a file.
    We can create as many ids as we want.
    Syntax:
    <head>
    <style>
    #red_font{ <--- hash<id _name>
    color:red;
    }
    </style>
    </head>

    <body>
    <p id="red_font">Hello Mr. DJ</p>
    <p>How are you?</p>
    </body>

    RESULT:

    Hello Mr. DJ

    How are you?


  3. ATTRIBUTE Selector: This is used to give certain elements having certain attribute, css properties.
    Syntax:
    <head>
    <style>
    p[draggable]{ <--- element[attribute or attribute="value"]
    color:red;
    }
    </style>
    </head>

    <body>
    <p draggable="TRUE">Hello Mr. DJ</p>
    <p>How are you?</p>
    </body>

    RESULT:

    Hello Mr. DJ

    How are you?


  4. UNIVERSAL Selector: This is used to give all elements css properties.
    Syntax:
    <head>
    <style>
    *{ <--- asteric
    color:red;
    }
    </style>
    </head>

    <body>
    <p draggable="TRUE">Hello Mr. DJ</p>
    <p>How are you?</p>
    </body>

    RESULT:

    Hello Mr. DJ

    How are you?
